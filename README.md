# mirai-login-solver-sakura

--------------------------

这是另外的一整套验证处理工具，主要是为了优化和方便处理各种验证码

> 本项目前身: [TxCaptchaHelper](https://github.com/mzdluo123/TxCaptchaHelper)



## 系统要求

| mirai-login-solver-sakura | mirai-core | Android             |
|:--------------------------|:-----------|:--------------------|
| 0.0.1 - dev               | 2.13.0+    | Android 9+ (API 28) |

--------------
